# My Portfolio Website

This is my personal portfolio website where I showcase my skills, projects, and contact information.  
The website is built using **HTML, CSS, and JavaScript**, and it uses **Google Sheets as a database** to store form data.

---

## 🚀 Features

- Responsive and modern portfolio design
- AboutMe section to introduce myself
- My Work section to showcase my work
- Contact form for visitors
- Form data stored directly in Google Sheets
- Smooth UI using CSS and JavaScript

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and form handling  
- **Google Sheets** – Used as a simple database for storing form submissions

## 📊 Database Integration

The contact form on the website is connected to **Google Sheets**.  
When a user submits the form:

1. The data is captured using **JavaScript**
2. It is sent to **Google Sheets using a Web App Script**
3. The responses are stored automatically in the sheet

This allows easy viewing and management of contact messages.
